## Installation procedure for android terminals( TERMUX )
```
>> apt update 
>> apt upgrade 
>> pkg istall python-y 
>> pkg install python3 
>> pip install art
>> pip install git 
>> git clone https://github.com/harkerbyte/font-generator
>> cd font-generator
>> python3 cy-art.py

```
## Installation for kali

* sudo git clone https://github.com/shade234sherif/font-generator
* cd font-generator 
* python3 cy-art.py
